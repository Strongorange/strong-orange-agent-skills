# strong-orange-agent-skills

Personal skill package repository for `npx skills`.

## Install

```bash
npx skills add Strongorange/strong-orange-agent-skills --all
```

Install specific skills:

```bash
npx skills add Strongorange/strong-orange-agent-skills --skill agent-handoff --skill functional-spec-author --skill strategy-template-governor
```

List available skills without installing:

```bash
npx skills add Strongorange/strong-orange-agent-skills --list
```
